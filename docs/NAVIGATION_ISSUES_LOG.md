# Navigation Issues Log

**Track navigation bugs and discrepancies.** When a button or flow sends the user to the wrong place, log it here. Update when issues are found or fixed.

---

## Governance

| Rule | Description |
|------|-------------|
| **Log on find** | When you discover a navigation bug (wrong destination, unexpected back behavior), add a row to the Issues table. |
| **Update on fix** | When an issue is fixed, set Status to `fixed` and add a Change log entry. |
| **Update change log** | When flows change or issues are logged/fixed, add an entry to the Change log section. |
