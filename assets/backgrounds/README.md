# Background Assets

Place the four time-of-day background images here.

| Filename         | Time Window | Description                        |
|------------------|-------------|------------------------------------|
| `dawn.png`       | 5am – 10am  | Warm amber sunrise, campfire glow  |
| `midday.png`     | 10am – 5pm  | Bright, high-contrast daylight     |
| `sunset.png`     | 5pm – 8pm   | Deep orange-red dusk               |
| `night.png`      | 8pm – 5am   | Deep navy, stars, fire as primary  |

**Style:** Refined Architectural Luxury. Think: illustrated landscape, textured,
painterly quality. Not photographic. Not flat/minimalist.

Until final assets arrive, the app uses `DayPeriodTheme` color gradients as placeholders.
