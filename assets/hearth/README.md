# Hearth (fire pit) assets

Watercolor fire pit images for the Sanctuary. Asset choice is driven by stones dropped this session (`hearthDropCount`):

| Filename | Use |
|----------|-----|
| `Hearth_Sizzle.png` | 0 stones dropped — low burning fire |
| `Hearth_High.png` | 1–2 stones dropped — medium fire |
| `hearth_extra_high.png` | 3+ stones dropped — huge fire |

Transparent or light-background PNGs work best so the forest background shows through. The app falls back to the circle+icon Hearth if an asset is missing.
